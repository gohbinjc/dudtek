DUDTEK Alpha

Durable vacuum monitoring for harsh wash‑down environments

Technical Specification

DUDTEK Alpha is a durable vacuum monitoring system designed to work in harsh wash‑down environments. The stainless‑steel machine can be set to monitor a wide range of metal closures after application on glass jars.

The machine is capable of monitoring whether the cap has been applied correctly and monitor the deflection of the metal closure to determine if a vacuum is present.
